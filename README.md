# Release Notes (EN 🇺🇸)

### Beta v0.1
#### Program logic

✅ Show the main menu:
- Ask the user what he wants to do:
  - Think of a number
  - Guess the number
  - Show the score
  - Exit the game

✅ The computer guesses a number:
- The rand() function generates a number
- The srand(time(0)) function, generates numbers every second and stores them in rand()

✅ The computer think of a number:
- The same mechanism as when guessing a number.
- Simulation of your "thinking"

✅ After each action
- Return the user to the main menu and ask them again what they want to do.

# Примечания к выпуску (RU 🇷🇺)

### Бета v0.1
#### Логика программы

✅ Показать главное меню:
- Спросить пользователя, что он хочет сделать:

  - Загадать число
  - Угадать число
  - Показать счёт
  - Выйти из игры

✅ Компьютер загадывает число:
- Функция rand() генерирует число
- Функция srand(time(0)), генерирует числа каждую секунду и сохраняет в rand()

✅ Компьютер угадывает число:
- Тот же механизм что и при загадывании числа.
- Симуляция вашего "загадывания"

✅ После каждого действия:
- Возвращать пользователя в главное меню и снова спрашивать, что он хочет сделать.
